## Usage
> Check out the [documentation](https://github.com/yaniswang/HTMLHint/wiki/Usage#how-to-use) to see the available options.

### Install

```a
npm install -D fly-htmlhint
```

### Example

```js
export default function* () {
  yield this
    .source("src/**/*.html")
    .htmlhint()
    .target("lib")
}
```
